# heic2jpg

ブラウザ上で動作する、HEIC/HEIF画像をJPEGに変換するシンプルなツールです。すべての処理はデバイス上で行われ、ファイルがサーバーにアップロードされることはありません。

## デモ

[https://code4fukui.github.io/heic2jpg/](https://code4fukui.github.io/heic2jpg/)

## 機能

-   **クライアントサイドでの変換:** HEICファイルをブラウザ内で直接JPEGに変換します。ファイルはサーバーに送信されないため、プライバシーが保護されます。
-   **バッチ処理:** 複数のHEICファイルをドラッグ＆ドロップして、一度にすべて変換できます。
-   **複数画像への対応:** 複数の画像を含むHEICコンテナを正しく処理します。
-   **自動ZIPダウンロード:** 複数の画像を変換した場合、1つの `.zip` アーカイブとして自動的にダウンロードされます。
-   **画像のリサイズ:** アスペクト比を維持したまま、最大幅または最大高さを設定して画像をリサイズできます。
-   **品質調整:** 出力されるJPEGファイルの圧縮率を調整できます。
-   **カラースペース管理:** Display-P3から、より互換性の高いsRGBカラースペースへ強制的に変換するオプションがあります。

## 使い方

1.  [heic2jpgのデモページ](https://code4fukui.github.io/heic2jpg/)を開きます。
2.  1つ以上の `.heic` ファイルをドロップゾーンにドラッグ＆ドロップします。
3.  （オプション）変換設定を調整します:
    *   **最大幅/高さ (px):** 出力画像の最大サイズを設定します（デフォルト: `1220`）。
    *   **品質 (0-1.0):** JPEGの品質を調整します（デフォルト: `0.9`）。
    *   **sRGBを強制:** チェックを入れると、カラースペースをsRGBに変換します。
4.  変換が即座に開始されます。処理が完了すると、ファイルが自動的にダウンロードされます。

## 必要条件

-   モダンWebブラウザ
